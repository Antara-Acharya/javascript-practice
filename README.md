Simple Todo List App

This is a basic and beginner-friendly Todo List web application made using HTML, CSS, and JavaScript.

Features

* Add new tasks
* Delete tasks
* Mark tasks as completed (Done/Undo)
* Simple and clean design
* Works instantly without page reload

Technologies Used

* HTML
* CSS
* JavaScript

Project Structure

index.html   (Main HTML file)
style.css    (Styling file)
todo.js      (JavaScript logic)

How It Works

1. Enter a task in the input box
2. Click the Add button
3. Task will appear in the list
4. Use Delete button to remove task
5. Use Done button to mark task as completed

How to Run

1. Download or clone the project
2. Open index.html in your browser
